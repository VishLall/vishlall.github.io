About project...

This is an Appointment Scheduling application that requires the user to login with the proper credentials before allowing the user to add new customers to a MySQL database, as well as update and delete. The user will be able to do the same with Appointments as they can with Customers. New appointments can be added, updated, and deleted from the same MySQL database. While logging in, the user receives a pop-up notification of any appointments within the next 15 minutes. There is also a Reports section where the user can see how many appointments were made in total by month and type. A logger has been implemented to track successful and unsuccessful log in attempts.
