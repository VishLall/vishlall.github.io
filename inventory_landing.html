About Project...

The user can add, update, and delete parts in this Inventory Management application. There's also a search function for finding parts and products. On the other hand, the user can make a product out of the many parts available. As a result, the user will create a product and then add the necessary parts to create it. The user will also have the ability to create, update, and delete such products. There are exceptions built in, so the user won't be able to delete a product until all of the parts that go with it have been deleted beforehand.
