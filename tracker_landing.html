About project...

This School Term Tracker for Android keeps track of the user's terms and courses. Users will be able to enter terms with beginning and ending dates. Then they'll be able to add all of the courses they'll be taking that term, as well as the instructors' contact information. After entering the courses, the user will be able to add upcoming assessments to each course and then enable notifications to know when their test is. Each assessment, course, and term can be updated and deleted by the user. However, there are exceptions in place that prevent the use from deleting a term if it contains classes. All information is saved and updated in a SQLite database.
